一个**极简至上**的仿**typecho**的WordPress主题，专注于内容本身，没有花哨的动画效果，没有复杂的交互功能。采用现代化的极简设计理念，让文字和图片成为视觉焦点，提供纯净、快速、高质量的阅读体验。内置Markdown支持、广告系统、友情链接等实用功能，开箱即用，无需复杂配置。

## 📦 安装

### 方法一：通过 WordPress 后台安装

1. 下载主题压缩包
2. 登录 WordPress 后台
3. 进入 `外观` → `主题` → `安装主题`
4. 上传主题压缩包
5. 激活主题

### 方法二：通过 FTP 安装

1. 下载主题压缩包并解压
2. 使用 FTP 工具连接到您的服务器
3. 将解压后的主题文件夹上传到 `/wp-content/themes/` 目录
4. 在 WordPress 后台激活主题

## ⚙️ 配置

安装并激活主题后，您可以在 WordPress 后台 `外观` → `barepaper主题设置` 中进行配置：

### 模块设置

在 `模块设置` 标签页中，您可以：

- ✅ **最新文章**: 显示或隐藏最新文章模块
- ✅ **标签云**: 显示或隐藏标签云模块
- ✅ **搜索**: 显示或隐藏搜索模块
- ✅ **分类**: 显示或隐藏分类模块
- ✅ **归档**: 显示或隐藏归档模块

其他模块功能正在开发中。

### 编辑器设置

- ✅ **经典编辑器**: 启用经典编辑器（推荐）
- ✅ **WDD-tinymce插件**: 启用 Markdown 语法支持

## 📝 使用说明

### 短代码

主题支持多种短代码，包括：

- `[video]` - 视频播放
- `[code]` - 代码块
- `[alert]` - 提示框
- `[button]` - 按钮
- `[quote]` - 引用
- `[music]` - 音乐播放器
- `[book]` - 书籍信息

详细用法请参考主题文档。

### Markdown 支持

主题支持 Markdown 语法，启用 WDD-tinymce 插件后，可以在编辑器中直接使用 Markdown 语法编写内容。
