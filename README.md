一个**极简至上**的仿**typecho**的WordPress主题，专注于内容本身，没有花哨的动画效果，没有复杂的交互功能。采用现代化的极简设计理念，让文字和图片成为视觉焦点，提供纯净、快速、高质量的阅读体验。内置Markdown支持、广告系统、友情链接等实用功能，开箱即用，无需复杂配置。


## 📋 主题信息


- 极简的风格，仿typecho默认主题；

- 侧栏的各种丰富的模块，文章==排行榜==，==置顶==文章，推荐文章，图文动态相册，归档，分类搜索等等，可以==自定义启用==（10.15-10.24）；
- 主题自动切换，暗主题和亮主题各元素适配(10.15)；
- 动态背景和个性鼠标等效果(10.15)；
- 友情链接系统，可以自定义链接(10.16)；
- 广告系统等，在页面顶部、底部和侧栏都有广告位，并改进广告标识防止被屏蔽(10.16)；
- ==书单==页面，可以分享自己的书籍，并根据在读、想读和已读分组(10.18)；
- 统计系统，如字数统计，在线统计等(10.19)；
- ==歌单==页面，能够分享在听的音乐，支持网易云音乐，QQ音乐和自定义音乐，支持==单曲==和==列表==播放(10.22)；
- 文章==AI自动摘要==，配置AI接口，实现自动生成文章摘要(10.23)；
- 改进编辑器，支持==markdown==，shortcode等语法,
- 优化模块，提升性能，针对高低性能的服务专门优化(10.26)；
- 缓存清理模块(10.28)；
- seo特别优化，增加rss和站点地图等功能，方便更好地被搜索引擎抓取(10.28)；
- 友好的移动端显示；
- 增加Cloudflare R2免费图床功能（11.27）
- 增加ip归属地和ua标识（11.29）
- 增加多图模式（11.30）；
- 禁用官方自带不需要的模块，主要是修订版本、代码标点转换、加载谷歌字体、离线编辑器开放接口、emoji载入css和js（11.30）；
- 上传图片重命名成时间戳 + 随机数（11.30）；
- 隐私控制，移除 Gravatar、s.w.org、版本号等控制，外部链接自动新窗口打开 (UX & SEO)（11.30）；
