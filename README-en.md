A **minimalist-first** WordPress theme inspired by **Typecho**, focusing on content itself without fancy animations or complex interactions. Adopting a modern minimalist design philosophy, making text and images the visual focus, providing a pure, fast, and high-quality reading experience. Built-in Markdown support, advertising system, friend links and other practical features, ready to use out of the box without complex configuration.

## 📦 Installation

### Method 1: Install via WordPress Admin

1. Download the theme zip package
2. Log in to WordPress admin
3. Go to `Appearance` → `Themes` → `Install Themes`
4. Upload the theme zip package
5. Activate the theme

### Method 2: Install via FTP

1. Download and unzip the theme package
2. Connect to your server using FTP
3. Upload the unzipped theme folder to `/wp-content/themes/` directory
4. Activate the theme in WordPress admin

## ⚙️ Configuration

After installation and activation, you can configure the theme in WordPress admin `Appearance` → `Barepaper Theme Settings`:

### Module Settings

In the `Module Settings` tab, you can:

- ✅ **Latest Posts**: Show or hide the latest posts module
- ✅ **Tag Cloud**: Show or hide the tag cloud module
- ✅ **Search**: Show or hide the search module
- ✅ **Categories**: Show or hide the categories module
- ✅ **Archives**: Show or hide the archives module

Other module features are under development.

### Editor Settings

- ✅ **Classic Editor**: Enable classic editor (recommended)
- ✅ **WDD-tinymce Plugin**: Enable Markdown syntax support

### Effect Settings

Visual effect features are under development.

### Performance Optimization

Performance optimization features are under development.

## 📝 Usage Instructions

### Shortcodes

The theme supports various shortcodes, including:

- `[video]` - Video player
- `[code]` - Code block
- `[alert]` - Alert box
- `[button]` - Button
- `[quote]` - Quote
- `[music]` - Music player
- `[book]` - Book information

For detailed usage, please refer to the theme documentation.

### Markdown Support

The theme supports Markdown syntax. After enabling the WDD-tinymce plugin, you can use Markdown syntax directly in the editor to write content.
